## What I Am Building

### 🔮 Lumio — Smart Gallery and Media Manager
An intelligent Android gallery application built with React Native and Expo EAS. Lumio goes beyond just displaying photos — it actively organises and cleans your media library.

**What it does:**
- Detects blurry, duplicate, and similar photos automatically
- Identifies burst shots, large files, screenshots, and short videos
- WhatsApp media cleanup with sent and received separation
- OCR-based document detection from photos
- Recycle bin with 30-day auto-delete
- Face detection and people grouping
- Image to PDF conversion with crop and rotation
- Text extraction from photos with export to PDF and DOCX
- Background scanning optimised for libraries of 5,000 to 20,000 photos

**Stack:** React Native · Expo EAS · SQLite · MMKV · ML Kit · React Navigation · FlashList

### 🏠 GrahShobha — Interiors and Furnishings Website
A modern website built for GrahShobha Interiors and Furnishings, a furniture manufacturing business based in Ujjain, Madhya Pradesh. The platform showcases furniture collections and interior solutions through a responsive, production-ready web experience.

Current Features

- Modern responsive UI built with Next.js
- Product catalogue and category browsing
- Product detail pages
- Per-product colour and size selection
- WhatsApp-based enquiry and ordering flow
- Animated and interactive user experience
- Deployed and accessible online

Tech Stack

- Next.js 14
- React
- Tailwind CSS
- Framer Motion
- Vercel

Currently Improving

- MongoDB integration for dynamic product data
- Cloudinary-based image management
- Product pricing and inventory management
- Admin dashboard for product management
- Authentication and authorization

🔗 Live on Vercel
